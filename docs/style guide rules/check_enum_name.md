### Check Enum Name
#### Description:
Enums should have capitalized names to make them clear and distinct from other types.

#### Incorrect Code:
```solidity
enum direction { UP, DOWN }
```

#### Correct Code:
```solidity
enum Direction { Up, Down }
```

#### Source:
[Enums - Solidity Style Guide](https://docs.soliditylang.org/en/v0.8.27/style-guide.html#enums)
